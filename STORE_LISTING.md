## Detailed Description

**SuperTables - The Ultimate Table Selection Tool for Chrome**

Tired of struggling to copy table data from websites? SuperTables brings Excel-like table selection to your browser, making it easy to select, copy, and analyze table data on any webpage.

**Key Features:**

- **Cell Selection**: Cmd/Ctrl + Click to select individual cells
- **Column Selection**: Alt + Click to select entire columns
- **Row Selection**: Cmd/Ctrl + Alt + Click to select entire rows
- **Multi-Selection**: Select multiple non-adjacent cells, rows, or columns
- **Range Selection**: Shift + Click to select rectangular ranges
- **One-Click Copy**: Selected data copies in Excel-compatible format
- **Statistics Panel**: Instantly see Sum, Average, Min, Max, and Count
- **Text Analysis**: Top 5 frequency analysis for text data
- **Excel Export**: Download entire tables as .xlsx files

**How It Works:**

1. Navigate to any webpage with tables
2. Hold Cmd/Ctrl and click on table cells to select them
3. Use the statistics panel to analyze your selection
4. Press Cmd/Ctrl + C to copy, paste directly into Excel

**Perfect For:**

- Data analysts extracting web data
- Researchers copying tables for reports
- Financial professionals analyzing web tables
- Anyone who works with online tables

**Privacy First:**
- No data collection
- No external servers
- All processing happens locally
- Open source

**Keyboard Shortcuts:**
- Cmd/Ctrl + Click: Select cell
- Alt + Click: Select column
- Cmd/Ctrl + Alt + Click: Select row
- Cmd/Ctrl + C: Copy selection
- Esc: Clear selection

Works on all websites with HTML tables. Supports nested tables and complex table structures.
